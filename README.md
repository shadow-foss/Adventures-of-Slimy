---

## If you get lost

```bash
git checkout game
git log --oneline
git checkout <commit-hash>
```

Each commit is named `stepX:` so you can jump to any point.

Return to latest version:

```bash
git checkout game
```

## Controls

* Move: `A / D` or Arrow keys
* Jump: `W` or Arrow Up
* Swap Color: `Tab` / `Space`
